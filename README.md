### 2. ElasticSearch 활용하기

(1) 인덱스 샤드 모니터링    
(2) 도큐먼트 CRUDS - 입력, 조회, 수정, 삭제, 검색    
(3) Query DSL: 풀텍스트 (Full Text), Bool, Range    
(4) 데이터 색인과 텍스트 분석  
(5) 노리(nori) 한글 형태소 분석기  
(6) 인덱스 매핑과 데이터 타입 - 문자열, 숫자, 날짜, 객체, 위치정보
